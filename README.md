# Spring Feign Client
Dois projetos para testar o Spring Feign client, que facilita o acesso a outras API REST.

Bastantes simples, acabei não escrevendo nenhuma classe de configuração, por exemplo para pegar o bearer da head.

Para poder subir ambos projetos ao mesmo tempo e testar, alterei a porta de um dos projetos.

- http://localhost:8091/v1/hello
- http://localhost:8080/feign/hello
